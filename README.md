# PERSONAL_DAILY_JOURNAL_APP_PROJECT
A console-based Journal Management System built in Python that allows users to write, store, manage, and track daily journal entries. The application saves data in a text file and provides full CRUD functionality (Create, Read, Update, Delete) for journal entries.

Features
1. Add new journal entries with automatic date stamping
2. View all saved journal entries in a structured format
3. Search entries by specific date
4. Edit existing entries for a selected date
5. Delete entries with user confirmation
6. File-based storage using `journal.txt`

Key Concepts Used
1. File handling (read, write, append modes)
2. Date and time handling using `datetime`
3. String manipulation and validation
4. Loops and conditional logic
5. Error handling using try-except blocks
6. CRUD operations implementation

Purpose: This project demonstrates practical file handling and data management in Python by simulating a real-world personal journal system. It helps improve understanding of file-based storage and basic application design.

Tech Stack
1. Language: Python
2. Module: datetime
3. Storage: Text file (`journal.txt`)
